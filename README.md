# hear-disease-prediction
This app helps in predicting heart disease in a patient based on few parameters like age,BP, Max HR, and gender

### Tech Stack:
- **Model:** RandomForestClassifier (88% Accuracy)
- **Library:** Scikit-Learn
- **Web App:** Streamlit

### How to run:
1. Install requirements: `pip install -r requirements.txt`
2. Run app: `streamlit run app.py`
